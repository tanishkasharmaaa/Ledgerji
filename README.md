# Ledgerji

Mobile-first digital ledger and business management platform for small businesses.

---

## Overview

Ledgerji is a modern business management platform designed for local businesses and shop owners to manage:

* Customer records
* Transactions
* Payment tracking
* Daily business activity
* Invoices
* Business analytics

The goal of Ledgerji is to simplify traditional khata management with a clean and easy-to-use digital experience.

---

## Features

### Customer Management

* Add and manage customers
* Search customer records instantly
* View transaction history

### Transaction Tracking

* Record payments
* Track paid/unpaid transactions
* Maintain digital business records

### Dashboard & Analytics

* Daily sales overview
* Revenue insights
* Business activity tracking

### Invoice & Billing

* Generate invoices
* Share invoices digitally
* Download invoice PDFs

### Mobile-First Experience

* Responsive UI
* Optimized for small business users
* Simple and clean workflow

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/ledgerji.git
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

---

## Environment Variables

Create a `.env` file:

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
PORT=5000
```
